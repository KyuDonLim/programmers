# 점 찍기
---
## 📃 문제 설명
```
 좌표평면을 좋아하는 진수는 x축과 y축이 직교하는 2차원 좌표평면에 점을 찍으면서 놀고 있습니다. 
 진수는 두 양의 정수 k, d가 주어질 때 다음과 같이 점을 찍으려 합니다.

 원점(0, 0)으로부터 x축 방향으로 a*k(a = 0, 1, 2, 3 ...),
 y축 방향으로 b*k(b = 0, 1, 2, 3 ...)만큼 떨어진 위치에 점을 찍습니다.
 원점과 거리가 d를 넘는 위치에는 점을 찍지 않습니다.

 예를 들어, k가 2, d가 4인 경우에는 (0, 0), (0, 2), (0, 4), (2, 0), (2, 2), (4, 0) 
 위치에 점을 찍어 총 6개의 점을 찍습니다.

 정수 k와 원점과의 거리를 나타내는 정수 d가 주어졌을 때, 
 점이 총 몇 개 찍히는지 return 하는 solution 함수를 완성하세요.
```
---
## 🔔 초기 구상
```
 1. 반지름이 d 인 원의 내부의 점을 찾는 것임을 파악
 2. 반복문을 통해서 해당 줄에서 몇 개의 점이 가능한지 파악
 3. 반지름이 d 인 원 둘레의 점의 x 좌표보다 작거나 같은 정수 더하기
```
---
## 📖 풀이 코드
```
 class Solution {
    public long solution(int k, int d) {
        long answer = 0;
        long dPow = (long) d * d;
        for(int i = 0; i <= d; i += k){
            int cnt = (int) Math.sqrt(dPow - Math.pow(i,2))/k;
            answer += (cnt + 1);
        }
        return answer;
    }
 }
```
---
## 💡 구현 내용
```
 1. k 만큼 떨어져야하기 때문에 반복문의 증감식을 k로 설정
   - for(int i = 0; i <= d; i += k)
 2. 원의 공식(x^2 + y^2 = r^2)을 바탕으로 x좌표 찾기
   - int cnt = (int) Math.sqrt(dPow - Math.pow(i,2))/k;
```
